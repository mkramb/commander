# ui-commander

* TypeScript
* Sass, CSS modules
* Mocha, Chai, Sinon

### Usage

```
{
  "name": "my-new-app",
  "version": "0.1.0",
  "dependencies" : {
    "react": "^15.6.1",
    "react-dom": "^15.6.1"
  },
  "scripts": {
    "shell": "ui-commander",
    "build": "ui-commander build",
    "build:watch": "ui-commander build --watch",
    "build:prod": "ui-commander build --prod"
  }
}
```
